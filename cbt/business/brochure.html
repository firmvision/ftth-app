Exam Security You Can Trust

Four layers of anti-cheat protection ensure every result is a true reflection of student knowledge.

🖥 Fullscreen Lock

Exam runs in mandatory fullscreen mode. Exiting triggers an immediate warning and violation count.

🔍 Tab-Switch Detection

Any attempt to switch browser tabs or windows is detected and logged in real time.

🔀 Question Randomisation

Questions and answer options shuffled uniquely per student — no two students see the same order.

🎫 One-Time Session Tokens

Each exam session uses a cryptographic token — cannot be shared, forwarded, or reused by another student.
