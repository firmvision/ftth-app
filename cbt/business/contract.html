Both parties agree as follows:

1. DEFINITIONS

  1. "Platform" means the Firmvision CBT Exam Platform software, including backend API, database, web interface, and all associated files delivered under this Agreement.
  2. "Deployment Environment" means the Client's local area network server and/or cloud server on which the Platform is installed.
  3. "Go-Live Date" means the date the Platform is confirmed operational by both parties in writing.
  4. "Support Period" means [12/24] months from the Go-Live Date, as selected in Schedule A.

2. SCOPE OF SERVICES

Firmvision agrees to provide the following services, detailed in Schedule A:

  1. Design, development, and delivery of the CBT Platform per the approved proposal (Ref: FVT/CBT/2026/001).
  2. Installation and configuration on the Client's designated server(s).
  3. Data migration / initial system population (if applicable).
  4. Staff training: administrator, teacher, and IT staff sessions.
  5. Post-launch support and maintenance for the Support Period.
  6. Documentation: user guides, admin handbook, and IT setup guide.

3. CLIENT OBLIGATIONS

  1. Provide a suitable server with specifications meeting Firmvision's minimum requirements (detailed in the IT Setup Guide).
  2. Designate a System Administrator ("Admin Contact") responsible for day-to-day operation.
  3. Ensure reliable power supply and local area network (LAN) connectivity across examination halls.
  4. Provide timely feedback and approvals at each project milestone.
  5. Not modify, resell, sublicense, or reverse-engineer the Platform software.
  6. Maintain confidentiality of administrator credentials and exam content.
